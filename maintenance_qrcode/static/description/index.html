Maintenance QR Code

Generate, preview, and print QR labels for maintenance equipment

This module extends Odoo Maintenance to provide equipment codes, QR generation, and printable QR labels for fast equipment lookup.

Features

  • Auto-generate internal equipment code from sequence
  • Manual code edit for Equipment Manager
  • QR payload opens equipment form directly
  • QR preview in Equipment form
  • Batch print labels from list/form actions
  • Label size presets: A4 (3x8), 50x30 mm, 70x50 mm

How to use

  1. Go to Maintenance > Equipment.
  2. Create equipment or open existing equipment.
  3. Generate/Edit Equipment Code.
  4. Use Action > Print QR Code.
  5. Choose label size and copies, then print.
