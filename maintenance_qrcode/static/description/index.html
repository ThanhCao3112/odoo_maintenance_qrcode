Maintenance QR Code

Generate and print QR labels for maintenance equipment

Maintenance QR Code extends Odoo Maintenance to add internal equipment codes, QR preview, and printable QR labels for fast asset identification and lookup.

Key Features

  • Auto-generate Equipment Code from sequence.
  • Manager-only button to generate missing codes.
  • Unique code validation to avoid duplicates.
  • QR payload opens equipment form directly.
  • QR preview on equipment form.
  • Label print wizard with formats: Dymo, 2 x 7, 4 x 7, and Custom Columns.
  • Configurable label header color.

How It Works

  1. Go to Maintenance > Equipment.
  2. Create equipment or open an existing record.
  3. Generate equipment code if needed.
  4. Click Equipment QR Label in the equipment view.
  5. Select layout and quantity, then print PDF labels.

Security

  • Only users in Equipment Manager group can generate equipment codes.
  • Standard Maintenance access rights remain unchanged.
